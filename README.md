# Swiggy Restaurant Discovery Case Study

## Problem
Many users repeatedly see the same restaurants on food delivery apps.

## Objective
Understand user ordering behavior and propose a feature that helps users discover new high-rated restaurants.

## Method
- User survey
- Data analysis
- Product solution design
